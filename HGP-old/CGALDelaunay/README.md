# CGAL Order-K Delaunay

This directory contains the C++ implementation of the Order-K Delaunay triangulation.

## Structure
- `orderk_delaunay_cpp/`: Source code for the C++ tool.

## Build
Run `python3 scripts/setup_cgal.py` to build the tool.
The binary will be located at `orderk_delaunay_cpp/build/orderk_delaunay_cpp`.
