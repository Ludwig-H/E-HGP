# PERG-HGP: Progressive Entropic Rank-Gabriel HGP

`PERG-HGP` is a research prototype for progressive, entropy-regularized hypergraph percolation clustering on 3D point clouds.

The current PyTorch implementation constructs and post-processes a finite atlas generated by entropic critical witnesses. It has not been validated end to end at 30 million points, and it does not certify that the generated atlas is complete for the full power-HGP hierarchy.

See [Rapport_revue_complete_PERG_HGP_2026-07-10.md](Rapport_revue_complete_PERG_HGP_2026-07-10.md) for the mathematical contract, code audit, corrected guarantees, scalability analysis, and recommended 30M-point architecture.
